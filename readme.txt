Для запуска приложения 

1. Клонируем репозитории "git clone https://github.com/kethisXD/project_kt.git" или скачиваем его
2. Создаем виртуальное окружение "python -m venv .venv"
3. Активируем окружение если автоматически не активировался "venv\Scripts\activate"
4. Устанавливаем зависимости "pip install -r requirements.txt"
5. Тестируем
